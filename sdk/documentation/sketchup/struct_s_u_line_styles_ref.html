related

Retrieves the line styles provider given a name.

Since
SketchUp 2019, API 7.0
Parameters
[in]line_stylesThe line_styles manager object.
[in]nameThe name of the line style object to get. Assumed to be UTF-8 encoded.
[out]line_styleThe line style object retrieved.
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_INVALID_INPUT if line_styles is not a valid object
  • SU_ERROR_NULL_POINTER_INPUT if name is NULL
  • SU_ERROR_NULL_POINTER_OUTPUT if line_style is NULL
  • SU_ERROR_NO_DATA if name does not match the name of any existing style.
