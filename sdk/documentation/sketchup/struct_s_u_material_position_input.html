Detailed Description

SUMaterialInput contains information that is needed to apply a material to a face.

The conventional method for applying a material to a face is to use 1 to 4 UV coordinates, which are Cartesian textures coordinates and corresponding 3D points on the face's plane. If the 3D points are not on the face's plane they will be projected onto it. Once the material input is used (e.g. with SUEntitiesFill()), the material object must not be released since it will be associated with a parent object.

Since
SketchUp 2021.1, API 9.1
